<h1>What is this About ?  </h1>
Project on Simple authentication using GraphQL and Node.js 
The project contains a login and signup form upon which the user can be authenticated.  

<h6>Tech Used</h6>
<ul>
<li> React </li>
<li> GraphQl </li>
<li> Node js </li>
<li> Express js </li>
<li> MongoDb</li>  
</ul>
